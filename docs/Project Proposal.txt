What Our App Will Do: 
Our project will help users identify movies/TV shows from partial or remembered quotes. Users can enter a line 
of dialogue, and the app will search through a movie script dataset, rank potential matches using a text 
similarity algorithm, and display the most likely results. Each result is clickable. When a user selects a movie, 
the app fetches detailed information such as ratings, a plot summary, and the movie poster from the TMDb 
API. This application will include dynamic routes, interactive stateful components, and a responsive, polished 
user interface. 
 
API Link: 
https://developer.themoviedb.org/docs/getting-started 
 
API Description: 
We will be using the TMDB (The Movie Database) API, a RESTful web service. Our app will primarily call GET 
endpoints such as “/3/search/movie” (title-based search) and “/3/movie/{movie_id}” (detailed movie lookup) to 
retrieve movie metadata in JSON format. 
 
When our quote-matching system produces ranked candidate movies, we will resolve each candidate to a 
TMDB movie_id (either by searching title via “/search/movie” or, if we have an IMDb ID, by using TMDB’s “Find 
By ID” endpoint “/3/find/{external_id}”). 
 
The API returns data such as title, release date/year, overview (plot summary/description), genres, and 
vote_average (TMDB’s rating-like score), along with poster_path values. To display posters, we will construct 
full image URLs using TMDB’s documented image URL process (base URL + size + file path). 
 
We will use this metadata to populate a /movie/:tmdbId detail page when a user selects a ranked quote match. 
Our application will send the selected movie’s TMDB ID (and our API key / auth token) as part of the HTTP 
GET request, and it will use the structured JSON response to render the movie’s information into the UI. No 
persistent data will be sent back to TMDB aside from the read-only lookup requests. 
 
UI Organization Description: 
Routes / Pages: 
●  Home (/): Hero section with app name, tagline, and a centered quote search bar. Includes a 
navbar with links to Search and FAQ. (option: a small text or FAQ to guide/help new users) 
●  Search (/search): Full quote input with optional filters — Genre, Year Range (years), and Sort 
By. Clean filter row beneath the search bar for quick refinement. 
●  Results (/results): Ranked list of matching movies displayed as cards. Each card shows a movie 
poster, title, release year, and a short matched quote snippet. Cards are clickable and navigate 
to the Movie Details page. 
●  Movie Details (/movie/:tmdbId): Full movie info pulled from TMDB (poster, title, plot summary, 
genres, rating, and release year).  
 
Main interactions: 
Submit quote → fetch matches → display ranked results 
Click a result → navigate to the details page and fetch full metadata
